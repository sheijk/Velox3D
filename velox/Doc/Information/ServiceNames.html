Velox Service Names

Each subsystem/service has a unique prefix. The names of service objects are composed of the prefix followed by a dot and an abbreviation for the object's type

Virtual File System

prefix: vfs
  • vfs.fs - IVFileSystem
  • vfs.dpp - IVDataProviderPool
  • vfs.strfact - IVStreamFactory

Debug/Error/Logging

prefix: error.service

Editor

prefix: edit

XML Service

prefix: xml.service

Graphics Service

prefix: graphics.service

Config Service

prefix: config.provider

Console Service

prefix: console.service
