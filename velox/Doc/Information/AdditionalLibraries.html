Additional Libraries

Velox3D uses additional libraries that are not part of the actual build

List of all non velox API's:

  • wxWindows - Usage of the GUI framework
  • TinyXML - Basic xml parser
  • Boost - Used in virtual filesystem
  • extgl - by graphics service for loading gl extensions

The additional libraries cannot be found in the CVS. If you add a new library make sure you packed all project files into a zip onto the ftp server (the actual lib file in debug and release mode too) and add a short reference on how it is going to work if needed.

External Lib Structure:

Debug libs are marked with a "d" after its name.

  • Local\ExternalLibs\Name of library(d)
