Velox Goals

Velox3D will be an extensible general purpose 3D Engine Framework. The focus lays on building a solid base for future productions of Sechsta-Sinn.

Editor

Building a framework for all Velox related editors. The goal is a shared IDE for developing games with velox. Possible editors could be geometrie, world builder, script editor, etc.

Graphics Engine

It should be possible to include different kinds of geometry representation using each ones

Game Engine

A big game world including geometry and entities shall be possible
